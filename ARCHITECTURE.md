# YeMind Zen 架构

## 项目定位

- 插件 ID：`yemind-zen`
- 首个源码重构版本：`0.1.0`
- 导图内核：`simple-mind-map`（MIT）
- 思源集成参考：`suka233/siyuan-kmind-plugin`（按其仓库许可证使用）
- 视觉与交互：采用用户确认过的上、左、下三组工具栏、Dock 和设置方案

## 迁移原则

1. 不再修改 KMind Zen 压缩包中的私有模块。
2. 不存在 Pro、付费、试用、激活、会员或能力开关。
3. 所有新代码进入 `src/`；根目录 `index.js` 是当前可安装运行包。
4. 每完成一个内核模块迁移，就删除旧运行时中对应实现，避免双轨逻辑。
5. 文件兼容通过 `src/compat/` 实现，不污染编辑内核。

## v0.1.0 范围

- 完成项目更名、插件 ID、存储 ID、标签页 ID、Dock ID 和协议链接迁移。
- 建立 TypeScript + Vite 源码工程。
- 建立 `simple-mind-map` 编辑内核入口。
- 保留上一版可运行构建作为过渡运行时，避免交付一个无法打开的空壳。
- 图标统一为绿色 `Ye`。
