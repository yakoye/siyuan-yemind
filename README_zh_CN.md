# YeMind Zen

YeMind Zen 是一个基于 `simple-mind-map` 构建的思源笔记思维导图插件。

- 插件目录和插件 ID：`siyuan-yemind-zen`
- 思源显示名：`YeMind Zen`
- 当前版本：`0.5.6`
- 本地保存，不包含付费、试用、激活或会员逻辑

## v0.5.6：现有功能回归与稳定性修复

本版不增加新功能，集中检查右键菜单、文字选区工具栏、节点工具、保存、新建、关闭、设置、拖拽、缩放、适配和视图重置。

- 修复只读模式和根节点保护遗漏。
- 修复多选后右键可能操作错误主节点的问题。
- 修复保存失败后内存状态提前变化、旧保存状态覆盖新修改等持久化问题。
- 修复恢复标签页加载失败后残留注册，以及重复关闭标签的问题。
- 统一节点链接和行内链接的协议校验与打开方式。
- 修复本地图片读取失败产生未处理异常的问题。
- 补齐菜单、选区、工具栏、设置、生命周期和画布操作的自动化回归检查。

拖拽、框选、缩放、适配视图、重置视图、布局和历史记录继续使用 `simple-mind-map` 原生实现。

## 当前功能

- Dock：新建、打开、复制链接、重命名、删除导图
- 节点编辑、添加、删除、结构调整、撤销和重做
- 同一导图节点子树复制、剪切和粘贴
- 富文本、公式、挖空、批注、待办、标签、图标、链接和图片
- 行内超链接、行内代码和代码块编辑器
- 概要、关联线、多种布局和自动保存
- 搜索、替换、大纲和分屏
- 只读、禅模式和全屏

## 安装

把整个 `siyuan-yemind-zen` 文件夹放入：

```text
工作空间/data/plugins/siyuan-yemind-zen/
```

重启思源后启用 **YeMind Zen**。

## 开发原则

新增功能前先完整分析 `simple-mind-map` 对应插件、命令、事件、数据字段和关联模块。能够调用原生能力时不增加第二套状态层；一个版本只处理一个小主题。

后续版本计划见 `DEVELOPMENT_PLAN.md`。
