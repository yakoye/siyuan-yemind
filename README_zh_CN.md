# YeMind

YeMind 是面向思源笔记的本地优先思维导图插件，支持导图、分屏大纲、纯大纲、富文本节点、图片、备注、批注、标签、待办、节点样式、整图主题、检查点和全局搜索。

当前版本：`0.9.3`  
开发基线：SiYuan `3.7.3`

## v0.9.3 更新

- 透明中心主题自动填充当前主题或项目背景色，分支线不再从标题文字下方透出。
- 节点无需点击选中，鼠标悬停即可显示 `+`、`−` 或子孙数量；移向按钮时不会提前消失。
- 分屏和纯大纲默认升级为连续文本编辑器，可像普通文本编辑器一样跨多个节点选择、复制、剪切、粘贴和替换。
- 每行对应一个节点，空格或 Tab 缩进表示层级；可直接粘贴书签、目录和外部大纲自动生成节点树。
- Tab / Shift+Tab 可批量调整所选行层级，Enter 自动延续缩进；文本变化通过一次可撤销整树事务同步。
- 中文输入法组合输入完成后才同步；完全改写标题仍保留原节点 UID、备注、标签、图片和局部样式。
- 保留“节点”模式，用于单节点富文本、拖动、展开和折叠；两种模式始终共享同一导图数据。

## v0.9.2 更新

- 19 套命名主题增加中心、一级、二级和普通节点的独立边框颜色。
- 主题和彩虹连线切换改为统一外观事务：先更新全部配置，再执行一次完整重绘。
- 切换主题、彩虹配色或宿主明暗模式后立即刷新，不再等待节点结构变化。
- 完整重绘保留当前缩放、平移和节点选择，节点局部文字、背景、边框和连线样式继续优先。
- 主题颜色与边框统一来自 `yemind_theme_colors_with_borders.json`，避免菜单、数据和实际渲染不一致。

## v0.9.1 更新

## v0.9.0 更新

- 节点图片右上角改为清晰的删除图标，删除前显示确认对话框。
- 节点图片左上角增加放大镜；点击后在当前编辑器内预览图片。
- 图片预览支持鼠标滚轮缩放、1:1 恢复、右上角关闭、点击图片外区域关闭和 Esc 关闭。
- 备注与批注悬停不再显示会遮挡内容的浏览器提示标签。

- 主题保留 YeMind 默认、Ink Branch、Material 3 Basic，并加入上述十套完整配色方案；每套包含背景色、节点色、文字色和分支色。

## 主要功能

- 导图、分屏和纯大纲三种视图。
- 节点新增、删除、折叠、拖动、跨层级调整与撤销重做。
- 画布和大纲富文本编辑，支持局部选区格式、链接、公式和代码块。
- 节点图片粘贴、拖入、等比例缩放、删除确认和沉浸预览。
- 长文备注、批注、待办、标签、图标、概要、外框与关联线。
- 单节点样式和整图密度、背景、彩虹分支、主题方案。
- 思源全局搜索中的节点结果、大纲预览和节点定位。
- 自动保存、检查点、恢复保护和结构化诊断包。

## 安装与覆盖升级

1. 完全退出思源。
2. 将发布 ZIP 内的文件直接解压到：
   `工作空间/data/plugins/siyuan-yemind/`
3. 覆盖同名文件后重新启动思源。

发布包不包含用户的 `maps.json`、`settings.json` 和 `checkpoints.json`。
