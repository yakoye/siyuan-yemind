# YeMind Zen

YeMind Zen 是一个基于 `simple-mind-map` 构建的思源笔记思维导图插件。

- 插件目录和插件 ID：`siyuan-yemind-zen`
- 思源显示名：`YeMind Zen`
- 当前版本：`0.5.5`
- 本地保存，不包含付费、试用、激活或会员逻辑

## v0.5.5：原生概要与关联线完善

本版只完善概要和关联线，不同时加入外框、导出或思源深度集成。

- 多选同一父节点下的节点后添加概要，范围计算直接使用 `simple-mind-map` 原生逻辑。
- 右键菜单会区分“添加概要”“为所选节点添加概要”“删除当前概要”和“删除该节点全部概要”。
- 创建关联线时显示轻量提示；按 `Esc` 或点击“取消”结束创建。
- 点击关联线后可以编辑文字、删除关联线，并直接拖动原生端点和贝塞尔控制点。
- 设置中增加默认概要文字、默认关联线文字、关联线层级和控制点开关。
- 保存前清理已删除目标留下的无效关联数据，同时保持端点、控制点、文字和样式索引一致。

概要范围、关联线绘制、文字编辑、端点、控制点、历史和渲染仍由 `simple-mind-map` 原生实现。

## 当前功能

- Dock：新建、打开、复制链接、重命名、删除导图
- 节点编辑、添加、删除、结构调整、撤销和重做
- 同一导图节点子树复制、剪切和粘贴
- 富文本、公式、挖空、批注、待办、标签、图标、链接和图片
- 行内超链接、行内代码和代码块编辑器
- 概要、关联线、多种布局和自动保存
- 搜索、替换、大纲和分屏
- 只读、禅模式和全屏

## 安装

把整个 `siyuan-yemind-zen` 文件夹放入：

```text
工作空间/data/plugins/siyuan-yemind-zen/
```

重启思源后启用 **YeMind Zen**。

## 开发原则

新增功能前先完整分析 `simple-mind-map` 对应插件、命令、事件、数据字段和关联模块。能够调用原生能力时不增加第二套状态层；一个版本只处理一个小主题。

后续版本计划见 `DEVELOPMENT_PLAN.md`。
