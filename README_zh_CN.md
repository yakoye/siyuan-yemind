# YeMind Zen

YeMind Zen 是一个基于 `simple-mind-map` 构建的思源笔记思维导图插件。

- 插件目录和插件 ID：`siyuan-yemind-zen`
- 思源显示名：`YeMind Zen`
- 当前版本：`0.5.1`
- 本地保存，不包含付费、试用、激活或会员逻辑

## v0.5.1：拖拽、布局与数据稳定性

### 结构化拖拽

默认恢复为“结构化拖拽”：

- 拖动父节点时，完整子树一起移动。
- 拖到其他节点上时成为其子节点。
- 拖到同级节点之间时自动重新排序。
- 松开后由导图内核重新计算上下节点和同级节点间距。
- 打开旧导图时会清除遗留的自由坐标，恢复正常树形排版。

仍可在设置中切换“自由拖拽”，但自由拖拽用于手工定位，不会自动保持标准树形间距。

### 防止画布飞走

- 默认关闭节点拖拽时的边缘自动平移。
- 拖拽开始时记录画布位置，结束后按设置恢复。
- 不保存拖拽过程中的临时画布位置。
- 对历史缩放和画布坐标进行范围校验，异常位置会被忽略。
- 恢复了上次视图时，不再被“打开时适配视图”重复覆盖。

### 未命名导图不再丢失

- 空标题导图会规范为“未命名导图”，不会在启动时被过滤。
- 新建、重命名、删除和 URL 打开操作会等待数据加载完成。
- 修复启动加载尚未结束时新建导图可能被旧数据覆盖的问题。

### 设置

设置现在分为：

- 常规
- 拖拽与布局
- 节点与内容
- 快捷键

新增设置包括：

- 结构化拖拽 / 自由拖拽
- 边缘自动平移
- 拖拽后保持视图位置
- 恢复上次视图位置
- 限制导图在画布内
- 最小和最大缩放
- 适配视图留白
- 二级节点横向、纵向间距
- 下级节点横向、纵向间距

左侧工具栏和节点右键菜单新增“整理布局”。

## 既有功能

- Dock：新建、打开、复制链接、重命名、删除导图
- 节点编辑、添加、删除、结构调整、撤销和重做
- 富文本、公式、挖空、批注、待办、标签、图标、链接和图片
- 行内超链接、行内代码和完整代码块编辑器
- 概要、关联线、多种布局、自动保存
- 搜索、替换、大纲和分屏
- 只读、禅模式和全屏

## 安装

把整个 `siyuan-yemind-zen` 文件夹放入：

```text
工作空间/data/plugins/siyuan-yemind-zen/
```

重启思源后启用 **YeMind Zen**。
