# YeMind

YeMind 是面向思源笔记的本地优先思维导图插件，支持导图、分屏大纲、纯大纲、富文本节点、图片、备注、批注、标签、待办、节点样式、整图主题、检查点和全局搜索。

当前版本：`0.9.22`  
开发基线：SiYuan `3.7.3`

## v0.9.22 原始 SVG 精确隔离

- 搜索、整图样式、撤销/重做、上级/同级/下级节点、节点样式、关联线、外框、图标、剪贴图和全屏等 14 个图标，逐字节使用 `图标-svg.txt` 中的 Base64 SVG，不再重写路径或颜色。
- 通过 `<img>` 文档边界承载 SVG，思源主题或自定义代码片段中的 `svg path { fill: ... }` 无法进入图标内部，避免线框路径被填成黑色实心块。
- YeMind 只控制外层统一的 `18 × 18` 占位和对齐，原图的宽高、描边、填充、虚线、遮罩和透明度均保持不变，显示效果与浏览器直接预览一致。
- 新增 SHA-256 原图契约、无依赖离线回归和恶意宿主 CSS 的真实 Chromium 回归。

- 主题保留 YeMind 默认、Ink Branch、Material 3 Basic，并加入上述十套完整配色方案；每套包含背景色、节点色、文字色和分支色。

## 主要功能

- 导图、分屏和纯大纲三种视图。
- 节点新增、删除、折叠、拖动、跨层级调整与撤销重做。
- 画布和大纲富文本编辑，支持局部选区格式、链接、公式和代码块。
- 节点图片粘贴、拖入、等比例缩放、删除确认和沉浸预览。
- 长文备注、批注、待办、标签、图标、概要、外框与关联线。
- 单节点样式和整图密度、背景、彩虹分支、主题方案。
- 思源全局搜索中的节点结果、大纲预览和节点定位。
- 自动保存、检查点、恢复保护和结构化诊断包。

## 安装与覆盖升级

1. 完全退出思源。
2. 将发布 ZIP 内的文件直接解压到：
   `工作空间/data/plugins/siyuan-yemind/`
3. 覆盖同名文件后重新启动思源。

发布包不包含用户的 `maps.json`、`settings.json` 和 `checkpoints.json`。
