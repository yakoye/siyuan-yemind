# YeMind Zen

YeMind Zen 是一个基于 `simple-mind-map` 构建的思源笔记思维导图插件。

- 插件目录和插件 ID：`siyuan-yemind-zen`
- 思源显示名：`YeMind Zen`
- 当前版本：`0.5.8`
- 本地保存，不包含付费、试用、激活或会员逻辑

## v0.5.8：检查点与安全恢复

本版增加独立于临时撤销栈的持久检查点：

- 顶部“检查点”菜单可创建命名检查点并打开管理窗口。
- 检查点保存节点、布局、主题和画布视图的完整深拷贝。
- 每张导图默认最多保留 20 个；超限只清理最旧普通检查点。
- 恢复历史前会自动创建“恢复前保护”检查点。
- 恢复通过主导图仓库事务写盘，再使用 `simple-mind-map.setFullData` 重新加载当前编辑器。
- 管理窗口支持查看时间与节点数、恢复、重命名和删除。
- 只读模式可查看和创建检查点，但不能恢复。

检查点保存在独立的 `checkpoints.json`，不会让每次导图自动保存都重写全部历史。

## 当前功能

- Dock：新建、打开、复制链接、重命名、删除导图
- 节点编辑、添加、删除、结构调整、撤销和重做
- 同一导图节点子树复制、剪切和粘贴
- 富文本、公式、挖空、批注、待办、标签、图标、链接和图片
- 行内超链接、行内代码和代码块编辑器
- 概要、外框、关联线、多种布局和自动保存
- 搜索、替换、大纲和分屏
- 只读、禅模式和全屏

## 安装

把整个 `siyuan-yemind-zen` 文件夹放入：

```text
工作空间/data/plugins/siyuan-yemind-zen/
```

重启思源后启用 **YeMind Zen**。

## 开发原则

新增功能前先完整分析 `simple-mind-map` 对应插件、命令、事件、数据字段和关联模块。能够调用原生能力时不增加第二套状态层；一个版本只处理一个小主题。

后续版本计划见 `DEVELOPMENT_PLAN.md`。
