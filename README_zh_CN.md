# YeMind Zen

YeMind Zen 是一个基于 `simple-mind-map` 构建的思源笔记思维导图插件。

- 插件目录和插件 ID：`siyuan-yemind-zen`
- 思源显示名：`YeMind Zen`
- 当前版本：`0.2.0`
- 图标：绿色底、白色 `Ye`
- 本地保存，不包含 Pro、付费、试用、激活或会员逻辑

## v0.2.0

本版已经完成运行内核的完整切换：根目录 `index.js` 和 `index.css` 均由 `src/` 通过 Vite 构建生成，不再使用 v0.1.0 的过渡运行时。

当前提供：

- 左侧 Dock：新建、打开、复制链接、重命名、删除导图
- 稳定标签页：同一张导图只打开一个标签页
- 节点双击编辑、添加子节点、添加同级节点、删除节点
- 节点拖拽调整层级
- 撤销、重做
- 画布平移、缩放、适配视图
- 原生右键菜单
- 向右、向左、双向、组织结构、目录组织等布局
- 自动保存、本地数据存储
- 只读模式、禅模式、全屏
- 顶部、左侧、底部三组轻量工具栏
- 基础设置：默认布局、画布操作、滚轮方式、节点快速入口、打开后适配视图

## 安装

把整个 `siyuan-yemind-zen` 文件夹放入：

```text
工作空间/data/plugins/siyuan-yemind-zen/
```

重启思源后启用 **YeMind Zen**。

旧目录 `yemind-zen` 不再使用，安装前请删除或改名停用，避免重复加载。

## 开发

```bash
npm install
npm test
npm run check
npm run build
```

源码位于 `src/`，测试位于 `tests/`。构建输出直接生成根目录的 `index.js` 和 `index.css`。
