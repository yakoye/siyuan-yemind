# YeMind Zen

- 当前版本：`0.5.20`
- 基于 `simple-mind-map` 的思源笔记导图插件

## v0.5.20：结构、节点样式与概要

本版增加节点尾端快速新增与折叠控件；导图、分屏和大纲统一使用原生 `expand` 状态；Root 本身保持固定，其一级分支可以正常折叠。顶部工具栏移除“删除”，将布局、主题和线型改成“结构”“主题”和线型图标，并把三项加入空白画布右键菜单。

节点右键菜单新增“节点样式”，可修改形状、填充、边框、宽度、字体、字号、字重、文字颜色、装饰和对齐。样式通过上游原生命令进入撤销历史和持久化。概要继续使用原生 `ADD_GENERALIZATION` 数据与括号布局，同时使用专用图标并在大纲中作为可编辑行显示。

## 开发资料

完整包包含 TypeScript 源码、测试、Superpowers 设计与实施计划、架构、功能矩阵、迁移说明和验证报告；发布包不包含 `node_modules/` 和临时 `dist/`。
