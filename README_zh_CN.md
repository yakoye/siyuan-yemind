# YeMind Zen

YeMind Zen 是一个基于 `simple-mind-map` 构建的思源笔记思维导图插件。

- 插件目录和插件 ID：`siyuan-yemind-zen`
- 思源显示名：`YeMind Zen`
- 当前版本：`0.3.0`
- 图标：绿色底、白色 `Ye`
- 本地保存，不包含 Pro、付费、试用、激活或会员逻辑

## v0.3.0

本版在 v0.2.0 的基础编辑内核上补齐节点内容能力。

### 富文本

双击节点进入编辑，选中文字后显示轻量工具栏，支持：

- 加粗、斜体、下划线、删除线
- 文字颜色、背景色
- 字号、字体
- 清除格式
- 挖空与取消挖空
- 行内公式和块级公式

挖空内容默认隐藏，鼠标移入时显示答案。公式使用 KaTeX 渲染并作为节点富文本保存。

### 节点内容

节点右键菜单新增独立入口：

- 待办：内容、完成状态、移除
- 批注：支持多条、新增、编辑和删除
- 备注
- 标签
- 图标
- 链接
- 图片：网络地址或本地图片，并支持调整尺寸
- 概要
- 关联线

待办和批注会在节点旁显示轻量状态标记，点击标记可直接查看和编辑。

## 基础功能

- 左侧 Dock：新建、打开、复制链接、重命名、删除导图
- 同一张导图复用同一标签页
- 节点编辑、添加、删除和拖拽调整层级
- 撤销、重做
- 画布平移、缩放和适配视图
- 多种布局
- 自动保存和本地数据存储
- 只读模式、禅模式和全屏
- 顶部、左侧和底部轻量工具栏

## 安装

把整个 `siyuan-yemind-zen` 文件夹放入：

```text
工作空间/data/plugins/siyuan-yemind-zen/
```

重启思源后启用 **YeMind Zen**。

## 开发

```bash
npm install
npm test
npm run check
npm run build
```

源码位于 `src/`，测试位于 `tests/`。构建输出生成根目录的 `index.js` 和 `index.css`。
