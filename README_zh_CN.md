# YeMind Zen

YeMind Zen 是一个基于 `simple-mind-map` 构建的思源笔记思维导图插件。

- 插件目录和插件 ID：`siyuan-yemind-zen`
- 思源显示名：`YeMind Zen`
- 当前版本：`0.5.2`
- 本地保存，不包含付费、试用、激活或会员逻辑

## v0.5.2：上游优先的稳定性收敛

本版不增加大功能，专门减少重复实现：

- 节点拖拽、完整子树移动、同级插入和布局重算全部交给 `simple-mind-map` 原生 Drag 与 Render 命令。
- 删除自制的拖拽视图捕获、拖拽结束恢复和临时视图拦截。
- 稳定路径固定为结构化拖拽，不再暴露自由拖拽实验选项。
- 思源恢复旧标签页时，先等待导图仓库和设置加载完成，再创建编辑器。
- 导图保存改为严格串行，避免旧保存晚完成后覆盖新数据。
- 关闭标签页前立即提交尚未触发的自动保存。

## 当前功能

- Dock：新建、打开、复制链接、重命名、删除导图
- 节点编辑、添加、删除、结构调整、撤销和重做
- 富文本、公式、挖空、批注、待办、标签、图标、链接和图片
- 行内超链接、行内代码和代码块编辑器
- 概要、关联线、多种布局和自动保存
- 搜索、替换、大纲和分屏
- 只读、禅模式和全屏

## 安装

把整个 `siyuan-yemind-zen` 文件夹放入：

```text
工作空间/data/plugins/siyuan-yemind-zen/
```

重启思源后启用 **YeMind Zen**。

## 开发原则

新增功能前先完整分析 `simple-mind-map` 对应插件、命令、事件、数据字段和关联模块。能够调用原生能力时不增加第二套状态层；一个版本只处理一个小主题。

后续版本计划见 `DEVELOPMENT_PLAN.md`。
