# YeMind Zen

YeMind Zen 是一个基于 `simple-mind-map` 构建的思源笔记思维导图插件。

- 插件目录和插件 ID：`siyuan-yemind-zen`
- 思源显示名：`YeMind Zen`
- 当前版本：`0.5.0`
- 本地保存，不包含付费、试用、激活或会员逻辑

## v0.5.0

### 待办与批注

节点右键菜单根据当前状态直接显示：

```text
没有待办 → 添加待办
待办未完成 → 待办完成
待办已完成 → 删除待办
```

操作立即执行，不再弹出待办窗口。批注使用清晰的线框备注图标，多条批注显示数量。

### 搜索、大纲和分屏

- 项目内搜索
- 上一个和下一个结果
- 当前替换和全部替换
- 导图视图
- 大纲视图
- 导图与大纲分屏
- 点击大纲节点定位到导图节点

### 完整设置

常规设置包括：

- 默认导图视图和布局
- 默认禅模式与只读模式
- 画布拖拽和滚轮行为
- 自动适配与自动保存
- 节点添加、菜单、待办和批注入口
- 行内链接、代码块和挖空

快捷键设置支持：

- 直接编辑
- 录制组合键
- 禁用
- 恢复默认
- 冲突检测

思源插件设置页和右上角 YeMind Zen 菜单会进入同一个完整设置窗口。

## 既有功能

- Dock：新建、打开、复制链接、重命名、删除导图
- 节点编辑、添加、删除、拖拽、撤销和重做
- 富文本、公式、挖空、批注、待办、标签、图标、链接和图片
- 行内超链接、行内代码和完整代码块编辑器
- 概要、关联线、多种布局、自动保存
- 只读、禅模式和全屏

## 安装

把整个 `siyuan-yemind-zen` 文件夹放入：

```text
工作空间/data/plugins/siyuan-yemind-zen/
```

重启思源后启用 **YeMind Zen**。
