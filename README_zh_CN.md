# YeMind Zen

YeMind Zen 是一个基于 `simple-mind-map` 构建的思源笔记思维导图插件。

- 插件目录和插件 ID：`siyuan-yemind-zen`
- 思源显示名：`YeMind Zen`
- 当前版本：`0.5.3`
- 本地保存，不包含付费、试用、激活或会员逻辑

## v0.5.3：原生节点子树剪贴板

本版只开发同一张导图内的节点复制、剪切和粘贴，不同时加入跨导图粘贴、外框或概要增强。

- `Ctrl/Cmd+C`：复制当前选中节点及其完整子树。
- `Ctrl/Cmd+X`：剪切当前选中节点及其完整子树；根节点由上游内核保护，不会被剪切。
- `Ctrl/Cmd+V`：把已复制或剪切的子树粘贴为当前节点的子节点。
- 多选时由 `simple-mind-map` 原生逻辑只保留最高层选中节点，避免父节点和后代同时复制后产生重复子树。
- 粘贴后的新节点 ID、激活状态、历史记录、重新布局和自动保存全部由原生命令处理。
- 使用上游已有的内存剪贴板路径，不依赖浏览器读取系统剪贴板的权限。
- 文本编辑状态仍由上游 `KeyCommand` 暂停画布快捷键，因此节点文字中的普通复制、剪切和粘贴不受影响。

当前范围是**同一张导图内**的节点子树剪贴板。跨导图粘贴和与外部应用交换节点数据不属于本版。

## 当前功能

- Dock：新建、打开、复制链接、重命名、删除导图
- 节点编辑、添加、删除、结构调整、撤销和重做
- 同一导图节点子树复制、剪切和粘贴
- 富文本、公式、挖空、批注、待办、标签、图标、链接和图片
- 行内超链接、行内代码和代码块编辑器
- 概要、关联线、多种布局和自动保存
- 搜索、替换、大纲和分屏
- 只读、禅模式和全屏

## 安装

把整个 `siyuan-yemind-zen` 文件夹放入：

```text
工作空间/data/plugins/siyuan-yemind-zen/
```

重启思源后启用 **YeMind Zen**。

## 开发原则

新增功能前先完整分析 `simple-mind-map` 对应插件、命令、事件、数据字段和关联模块。能够调用原生能力时不增加第二套状态层；一个版本只处理一个小主题。

后续版本计划见 `DEVELOPMENT_PLAN.md`。
