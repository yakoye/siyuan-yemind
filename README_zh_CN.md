# YeMind

YeMind 是面向思源笔记的本地优先思维导图插件，支持导图、分屏大纲、纯大纲、富文本节点、图片、备注、批注、标签、待办、节点样式、整图主题、检查点和全局搜索。

当前版本：`0.9.1`  
开发基线：SiYuan `3.7.3`

## v0.9.1 更新

- 完整接入 19 套主题：晨曦、彩虹、活力、舞动、代码、和风、岛屿、玫瑰、薄荷、绿茶、永恒、奶油、花海、珊瑚、绚丽、香槟、香水、禅心、律动。
- 每套主题分别控制导图背景、中心节点、一级节点、二级节点和普通节点的文字色与背景色。
- 中心到一级、一级到二级、二级到普通节点的连线颜色分别配置。
- 支持 1、3、4、6 色分支循环。
- YeMind 默认、Ink Branch、Material 3 Basic 三套基础主题均补齐浅色和深色配置。
- 主题菜单共 22 套主题，运行时包含 25 个实际外观定义。
- 主题颜色由 `docs/theme-colors/yemind_theme_colors.json` 单一数据源生成，构建、类型检查和测试前自动同步。
- 节点局部文字色、背景色和连线色继续优先于整图主题，主题生成样式不会写入导图数据。

## 主要功能

- 导图、分屏和纯大纲三种视图。
- 节点新增、删除、折叠、拖动、跨层级调整与撤销重做。
- 画布和大纲富文本编辑，支持局部选区格式、链接、公式和代码块。
- 节点图片粘贴、拖入、等比例缩放、删除确认和沉浸预览。
- 长文备注、批注、待办、标签、图标、概要、外框与关联线。
- 单节点样式和整图密度、背景、彩虹分支、主题方案。
- 思源全局搜索中的节点结果、大纲预览和节点定位。
- 自动保存、检查点、恢复保护和结构化诊断包。

## 安装与覆盖升级

1. 完全退出思源。
2. 将发布 ZIP 内的文件直接解压到：
   `工作空间/data/plugins/siyuan-yemind/`
3. 覆盖同名文件后重新启动思源。

发布包不包含用户的 `maps.json`、`settings.json` 和 `checkpoints.json`。
