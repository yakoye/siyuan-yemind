# YeMind Zen

YeMind Zen 是一个基于 `simple-mind-map` 构建的思源笔记思维导图插件。

- 插件目录和插件 ID：`siyuan-yemind-zen`
- 思源显示名：`YeMind Zen`
- 当前版本：`0.5.4`
- 本地保存，不包含付费、试用、激活或会员逻辑

## v0.5.4：原生框选与批量结构移动

本版只完善选择和多节点移动，不同时加入外框、概要增强或跨导图粘贴。

- 平移优先：左键拖动画布；按住 `Ctrl/Cmd` 后左键框选。
- 选择优先：左键直接框选；右键拖动画布。
- `Ctrl/Cmd + 单击节点`：把节点加入选择或从选择中移除。
- 拖动任一已选节点：由 `simple-mind-map` 原生 Drag 插件批量移动最上层所选子树。
- 父节点和后代同时被选中时，上游会过滤后代，避免同一子树被重复移动。
- 左侧“框”按钮用于快速切换平移优先和选择优先；底部只在多选时显示“已选 N”。

选择矩形、激活节点列表、批量拖拽、结构命令、历史记录和重新布局均由 `simple-mind-map` 原生实现，YeMind Zen 不维护第二套状态。

## 当前功能

- Dock：新建、打开、复制链接、重命名、删除导图
- 节点编辑、添加、删除、结构调整、撤销和重做
- 同一导图节点子树复制、剪切和粘贴
- 富文本、公式、挖空、批注、待办、标签、图标、链接和图片
- 行内超链接、行内代码和代码块编辑器
- 概要、关联线、多种布局和自动保存
- 搜索、替换、大纲和分屏
- 只读、禅模式和全屏

## 安装

把整个 `siyuan-yemind-zen` 文件夹放入：

```text
工作空间/data/plugins/siyuan-yemind-zen/
```

重启思源后启用 **YeMind Zen**。

## 开发原则

新增功能前先完整分析 `simple-mind-map` 对应插件、命令、事件、数据字段和关联模块。能够调用原生能力时不增加第二套状态层；一个版本只处理一个小主题。

后续版本计划见 `DEVELOPMENT_PLAN.md`。
