# YeMind Zen

YeMind Zen 是一个基于 `simple-mind-map` 构建的思源笔记思维导图插件。

- 插件目录和插件 ID：`siyuan-yemind-zen`
- 思源显示名：`YeMind Zen`
- 当前版本：`0.4.0`
- 本地保存，不包含付费、试用、激活或会员逻辑

## v0.4.0

### 行内超链接

在节点中选中文字后点击“链接”，可以添加、编辑或移除行内链接。支持：

- `http://`、`https://`
- 裸域名自动补 `https://`
- `mailto:`、`tel:`
- `siyuan://blocks/...`

外部链接可在设置中选择新窗口或当前窗口打开。

### 完整代码块编辑

富文本工具栏新增：

- 行内代码 `</>`
- 代码块

代码块窗口支持：

- 选择语言
- 新建和再次编辑
- Tab 插入 2/4 个空格
- 自动换行或水平滚动
- 调整代码字号
- 显示/隐藏语言标签
- 转为普通文本
- 删除整个代码块

### 设置

新增并实时生效：

- 自动保存延迟
- 富文本选区工具栏开关
- 裸域名自动补 HTTPS
- 外部链接打开方式
- 默认代码语言
- 代码块换行、语言标签、Tab 宽度和字号
- 挖空隐藏/模糊和悬停显示
- 待办与批注节点标记

## 既有功能

- Dock：新建、打开、复制链接、重命名、删除导图
- 节点编辑、添加、删除、拖拽、撤销和重做
- 富文本、公式、挖空、批注、待办、标签、图标、链接和图片
- 概要、关联线、多种布局、自动保存
- 只读、禅模式和全屏

## 安装

把整个 `siyuan-yemind-zen` 文件夹放入：

```text
工作空间/data/plugins/siyuan-yemind-zen/
```

重启思源后启用 **YeMind Zen**。

## 开发

```bash
npm install
npm test
npm run check
npm run build
```
