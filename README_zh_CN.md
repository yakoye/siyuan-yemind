# YeMind

YeMind 是面向思源笔记的本地优先思维导图插件，支持导图、分屏大纲、纯大纲、富文本节点、图片、备注、批注、标签、待办、节点样式、整图主题、检查点和全局搜索。

当前版本：`0.9.7`  
开发基线：SiYuan `3.7.3`

## v0.9.7 更新

- 向右逻辑图拖动改为“最近合法节点 + 放大局部区域”判定，适配节点宽高和文本长度不一致的真实导图。
- 每个目标节点左侧按中心线分为同级前插和后插区域，节点右半部及右侧扩展区表示成为该节点的子节点。
- 相邻节点之间使用最近距离、当前候选保持和切换迟滞，减少 54/65 一类密集节点区域中的来回闪动。
- 绿色虚线从拖动开始到结束始终存在；没有新目标时连接原父节点，靠近新区域后实时切换候选父节点。
- 父节点和拖动影子使用统一场景坐标绘制虚线，修复穿过 Root、错位及父节点与最终落点不一致。
- 绿色虚线、节点让位和最终移动共享同一个候选对象；保留单步撤销、Esc 取消以及完整子树元数据。

## v0.9.6 更新

- 大纲拖动触发区扩展为完整缩进单元格，绿色插入线采用整行上下区和横向层级吸附。
- 格式工具栏在继承主题字体时显示“默认字体”。
- 向右逻辑图移除画布插入线，以绿色候选父级虚线和节点让位表达预览。

## v0.9.5 更新

- 大纲结构拖动只从三角/方点左侧的隐形 gutter 发起，正文区域继续用于光标、文本编辑和跨节点选择；叶子方点缩小为 `5 × 5px`。
- 大纲拖动使用 YeMind 绿色、带方形起点的深度对齐插入线，明确显示同级前后、子节点和父级对齐位置。
- 画布拖动改为鼠标指针驱动的结构判定，拖动影子不再参与碰撞；两个节点之间的中性区域松手后维持原状。
- 画布拖动虚线始终连接原父节点或当前候选父节点，绿色插入线显示最终 BEFORE、AFTER 或 CHILD 落点。
- 同级候选快速响应，成为子节点需要短暂驻留；进入中性区立即清除旧候选，减少误挂、闪烁和残留目标。
- 拖动自己、后代、Root 非法位置和原位置均作为无效操作；Esc 可取消，一次有效移动只进入一条撤销记录。
- 图片删除按钮只缩小内部垃圾桶图形，使其与放大镜视觉重量一致；按钮背景尺寸和点击范围保持不变。
- 图片放大、删除和调整尺寸控件与节点结构拖动隔离，不会误触发节点移动。

## v0.9.4 更新

- 分屏和纯大纲不再区分“文本 / 节点”模式，统一为一个结构化节点编辑器；整棵树保持节点外观，同时支持原生跨节点文本选择。
- `Ctrl/Cmd+A` 第一次选择当前节点全部正文，第二次选择完整大纲；已有跨节点选区时直接提升为完整大纲。
- 粘贴遵循标准文本编辑器语义：选中文字后直接替换；支持单节点、跨节点、富文本、纯文本和缩进目录结构粘贴。
- 多行粘贴和结构编辑通过一次可撤销事务提交，保留边界节点 UID、备注、标签、图片、局部样式以及未被选区覆盖的原有子树。
- 分支节点使用纯黑三角，叶子节点使用等大的纯黑方点；继续保留 indent-rainbow 彩虹层级引导线。
- 大纲行完全扁平化：鼠标经过仅使用 `#ececec`，当前节点仅使用 `#deeae6` 和绿色文字，不显示边框、焦点框、阴影或左侧强调线。
- 节点拖动只从专用 gutter 发起，正文拖动始终用于选择文字；拖放时显示同级前后、父级对齐和子节点插入线。
- 选区格式工具栏在鼠标或键盘选择完成后出现，不在拖选过程中跳动；只读状态仍允许两级全选和复制。

## v0.9.3 更新

- 透明中心主题自动填充当前主题或项目背景色，分支线不再从标题文字下方透出。
- 节点无需点击选中，鼠标悬停即可显示 `+`、`−` 或子孙数量；移向按钮时不会提前消失。
- 分屏和纯大纲默认升级为连续文本编辑器，可像普通文本编辑器一样跨多个节点选择、复制、剪切、粘贴和替换。
- 每行对应一个节点，空格或 Tab 缩进表示层级；可直接粘贴书签、目录和外部大纲自动生成节点树。
- Tab / Shift+Tab 可批量调整所选行层级，Enter 自动延续缩进；文本变化通过一次可撤销整树事务同步。
- 中文输入法组合输入完成后才同步；完全改写标题仍保留原节点 UID、备注、标签、图片和局部样式。
- 保留“节点”模式，用于单节点富文本、拖动、展开和折叠；两种模式始终共享同一导图数据。

## 主要功能

- 导图、分屏和纯大纲三种视图。
- 节点新增、删除、折叠、拖动、跨层级调整与撤销重做。
- 画布和大纲富文本编辑，支持局部选区格式、链接、公式和代码块。
- 节点图片粘贴、拖入、等比例缩放、删除确认和沉浸预览。
- 长文备注、批注、待办、标签、图标、概要、外框与关联线。
- 单节点样式和整图密度、背景、彩虹分支、主题方案。
- 思源全局搜索中的节点结果、大纲预览和节点定位。
- 自动保存、检查点、恢复保护和结构化诊断包。

## 安装与覆盖升级

1. 完全退出思源。
2. 将发布 ZIP 内的文件直接解压到：
   `工作空间/data/plugins/siyuan-yemind/`
3. 覆盖同名文件后重新启动思源。

发布包不包含用户的 `maps.json`、`settings.json` 和 `checkpoints.json`。
