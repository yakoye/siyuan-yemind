# YeMind Zen

YeMind Zen 是一个基于 `simple-mind-map` 构建的思源笔记思维导图插件。

- 插件目录和插件 ID：`siyuan-yemind-zen`
- 思源显示名：`YeMind Zen`
- 当前版本：`0.5.7`
- 本地保存，不包含付费、试用、激活或会员逻辑

## v0.5.7：原生外框与多节点外框

本版严格接入 `simple-mind-map` 的原生 `OuterFrame` 插件，没有重新实现外框范围、坐标和历史记录。

- 单选普通节点可添加单节点外框。
- 多选连续同级节点可形成一个范围外框。
- 不连续同级节点自动拆成多个外框，不同父节点分别创建。
- 点击外框后可编辑文字、边框颜色、填充颜色、实线/虚线和文字对齐。
- 双击外框文字继续使用上游原生文字编辑器。
- 设置增加默认外框文字和横纵留白。
- 只读模式可查看外框，但不能添加、修改或删除。

范围分组、`groupId`、节点移动后的边界重算、删除、撤销重做和保存全部继续由上游内核处理。

## 当前功能

- Dock：新建、打开、复制链接、重命名、删除导图
- 节点编辑、添加、删除、结构调整、撤销和重做
- 同一导图节点子树复制、剪切和粘贴
- 富文本、公式、挖空、批注、待办、标签、图标、链接和图片
- 行内超链接、行内代码和代码块编辑器
- 概要、外框、关联线、多种布局和自动保存
- 搜索、替换、大纲和分屏
- 只读、禅模式和全屏

## 安装

把整个 `siyuan-yemind-zen` 文件夹放入：

```text
工作空间/data/plugins/siyuan-yemind-zen/
```

重启思源后启用 **YeMind Zen**。

## 开发原则

新增功能前先完整分析 `simple-mind-map` 对应插件、命令、事件、数据字段和关联模块。能够调用原生能力时不增加第二套状态层；一个版本只处理一个小主题。

后续版本计划见 `DEVELOPMENT_PLAN.md`。
