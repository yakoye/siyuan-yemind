# YeMind Zen

思源笔记中的独立思维导图插件。

- 插件目录：`yemind-zen`
- 当前版本：`0.1.0`
- 图标：绿色 `Ye`
- 导图内核迁移目标：`simple-mind-map`
- 不包含付费、Pro、试用、激活或会员功能

## 安装

把整个 `yemind-zen` 文件夹放入：

```text
工作空间/data/plugins/yemind-zen/
```

重启思源后启用插件。

## 开发

```bash
npm install
npm run dev
```

源码位于 `src/`。当前安装包暂时保留了上一阶段可运行内核，完整迁移状态见 `MIGRATION_STATUS.md`。
