# YeMind Zen

- Version: `0.5.20`
- SiYuan mind-map plugin based on `simple-mind-map`

YeMind Zen v0.5.20 adds compact node-tail add/fold controls, synchronized disclosure across map/split/outline views, Structure/Theme/Line project controls, blank-canvas project menus, a native node-style panel, and improved native summaries. Root remains fixed while first-level branches can fold. Node styles and summaries continue through upstream commands, history and persistence.

## Development contract

The archive contains complete TypeScript source, tests, Superpowers specifications and plans, architecture notes, feature matrix, migration notes and verification documentation. `node_modules/` and temporary `dist/` are excluded from release archives.

See `README_zh_CN.md` for the Chinese overview.
