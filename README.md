# YeMind Zen

YeMind Zen is a local-first mind-map plugin for SiYuan, powered by `simple-mind-map`.

- Plugin folder and ID: `siyuan-yemind-zen`
- Display name: `YeMind Zen`
- Version: `0.5.6`

v0.5.6 is a regression-only stability release covering context menus, rich-text selections, toolbar controls, transactional persistence, settings saves, restored tabs, link navigation, image loading, native drag, zoom, fit and view reset. No new feature surface was added.

See `README_zh_CN.md`, `FEATURE_MATRIX.md`, and `DEVELOPMENT_PLAN.md` for details.
