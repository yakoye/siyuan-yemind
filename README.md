# YeMind Zen

YeMind Zen is a local-first mind-map plugin for SiYuan, powered by `simple-mind-map`.

- Plugin folder and ID: `siyuan-yemind-zen`
- Display name: `YeMind Zen`
- Version: `0.5.9`

v0.5.9 adds an opt-in local diagnostics and regression mode: bounded sanitized logs, storage/map/checkpoint self-checks, global error capture, and ZIP export for real SiYuan runtime debugging. No diagnostics are uploaded automatically, and full map content is excluded unless explicitly selected.

See `README_zh_CN.md`, `FEATURE_MATRIX.md`, and `DEVELOPMENT_PLAN.md` for details.
