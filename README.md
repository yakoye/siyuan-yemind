# YeMind Zen

YeMind Zen is a local-first mind-map plugin for SiYuan, powered by `simple-mind-map`.

- Plugin folder and ID: `siyuan-yemind-zen`
- Display name: `YeMind Zen`
- Version: `0.5.5`

v0.5.5 completes summaries and associative lines through the existing native range-summary parser and `AssociativeLine` plugin. YeMind Zen adds only dynamic menu labels, a compact line-state toolbar, four native option settings, and immutable persistence cleanup for stale relation targets.

See `README_zh_CN.md`, `FEATURE_MATRIX.md`, and `DEVELOPMENT_PLAN.md` for details.
