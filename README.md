# YeMind Zen

YeMind Zen is a local-first mind-map plugin for SiYuan, powered by `simple-mind-map`.

- Plugin folder and ID: `siyuan-yemind-zen`
- Display name: `YeMind Zen`
- Version: `0.5.4`

v0.5.4 exposes native box selection, Ctrl/Cmd node toggling, and batch subtree movement through the existing `Select`, `MindMapNode`, and `Drag` implementations. YeMind Zen adds only a persistent mode toggle and a compact selected-node count; it does not maintain a second selection or drag state.

See `README_zh_CN.md`, `FEATURE_MATRIX.md`, and `DEVELOPMENT_PLAN.md` for details.
