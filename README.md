# YeMind Zen

YeMind Zen is a local-first mind-map plugin for SiYuan, powered by `simple-mind-map`.

- Plugin folder and ID: `siyuan-yemind-zen`
- Display name: `YeMind Zen`
- Version: `0.5.8`

v0.5.8 adds persistent named checkpoints in a separate transactional repository, protected pre-restore snapshots, and in-place restoration through `simple-mind-map.setFullData`. It does not add automatic scheduled backups, diffing, or a recycle bin.

See `README_zh_CN.md`, `FEATURE_MATRIX.md`, and `DEVELOPMENT_PLAN.md` for details.
