# YeMind Zen

YeMind Zen is a local-first mind-map plugin for SiYuan, powered by `simple-mind-map`.

- Plugin folder and ID: `siyuan-yemind-zen`
- Display name: `YeMind Zen`
- Version: `0.5.7`

v0.5.7 adds native single-node and continuous-sibling outer frames through `simple-mind-map`'s `OuterFrame` plugin, with a compact native-method toolbar and three upstream-backed settings. No custom frame geometry or second history layer is used.

See `README_zh_CN.md`, `FEATURE_MATRIX.md`, and `DEVELOPMENT_PLAN.md` for details.
