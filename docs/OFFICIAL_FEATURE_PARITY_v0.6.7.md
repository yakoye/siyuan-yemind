# YeMind Zen v0.6.7 官方功能适配边界

日期：2026-07-21

## 本版边界

- 整图背景色复用 YeMind 既有颜色面板与颜色解析，不建立第二套颜色状态。
- 全局搜索仍通过思源 `input-search` 插件事件扩展界面，不修改思源数据库或全文索引。
- 搜索结果优先插入原生结果容器顶部；点击只调用 YeMind 现有打开导图和延迟定位节点路径。
- 批注仅调整呈现布局，数据格式、编辑、删除和持久化语义不变。

## 保持不变

- `simple-mind-map` 继续负责导图结构、节点样式、历史与渲染。
- MapRepository 和 CheckpointRepository 的格式不变。
- v0.5.10 至 v0.6.6 的折叠、编辑、拖拽、富文本与搜索回归继续运行。
