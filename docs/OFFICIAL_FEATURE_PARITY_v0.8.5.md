# YeMind v0.8.5 分屏焦点事务适配边界

官方编辑器在单一状态树中协调画布与大纲焦点。YeMind 不复制官方私有状态内核，而是在现有 `simple-mind-map` 与 Quill 之间加入显式编辑表面协调器。

## 已适配

- 画布和大纲不能同时持有编辑恢复权；
- 画布交互会结束旧大纲编辑事务；
- 大纲结构命令后恢复目标节点；
- 外部结构变化不会复活旧大纲节点；
- 异步恢复请求可被新交互取消。

## 保持上游能力

节点插入、缩进、移动、删除、折叠、撤销和保存仍由 `simple-mind-map` 原生命令及 YeMind 命令适配层执行，不建立第二套树或历史系统。
