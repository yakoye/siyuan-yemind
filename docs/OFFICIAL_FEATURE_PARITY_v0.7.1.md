# YeMind v0.7.1 官方功能适配边界

v0.7.1 不修改 KMind 编辑器内核。本版修复的是思源全局搜索宿主适配：思源可能把关闭入口渲染为 SVG 或其他非 `HTMLElement` 元素，因此不能直接假定存在 `.click()`。

## 采用

- 结果列表和大纲预览继续嵌入思源原生搜索布局。
- Enter、双击和“打开导图”按钮继续进入 YeMind 原有标签打开与节点定位管线。
- 关闭控件通过能力检测激活，并提供 Escape 回退。

## 不采用

- 不修改思源原生搜索数据库和索引。
- 不替换思源搜索窗口实现。
- 不建立第二套标签页或节点定位机制。

## 数据兼容

无数据格式变化，无需迁移。
