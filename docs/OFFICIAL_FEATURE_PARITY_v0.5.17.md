# YeMind Zen v0.5.17 官方功能对齐边界

日期：2026-07-18

| 主题 | 官方 0.34.0 机制 | YeMind v0.5.17 | 边界 |
|---|---|---|---|
| 大纲正文 | 稳定行实体与持续编辑器 | UID 行 patch + 单活动 Quill | 不导入官方私有文档模型 |
| IME/提交 | 编辑事务与显式 flush | 组合期保护、160ms 提交、结构前 flush | 保存仍进入 YeMind 仓库 |
| 根节点 Enter | 创建子节点 | 已对齐 | 由上游插入命令执行 |
| 行拖拽 | handle/drop slot/结构事务 | 每行 handle，before/inside/after | 不实现跨画布与大纲双向拖拽协议 |
| 删除语义 | 正文删除与结构删除分离 | Backspace/Delete 仅编辑文字 | 节点删除仍走明确结构命令 |
| 节点图片 | 正文/大纲粘贴与比例缩放 | 画布节点粘贴/拖入并用上游等比例缩放 | 大纲行内图片编辑仍受当前 Quill/上游格式限制 |
| 备注 | 长文、图片、窗口缩放 | 独立备注、图片粘贴、尺寸保存 | 不导入官方备注文档内核 |
| 批注 | 多条时间线 | 保留多条批注并悬停预览 | 与备注分离 |
| 颜色 | 官方编辑器颜色控制 | 52 色、更多颜色、HEX/RGB 双向输入 | 用户指定交互优先 |
| 禅模式 | 右上角状态可识别并退出 | `● 禅` → `● 退出禅模式` | YeMind 自有胶囊样式 |
| 待办 | TODO 状态和删除 | 菜单添加/删除；复选框完成切换 | 不把“完成”作为菜单删除替代 |
| 诊断 | 官方未作为公开数据契约 | 本地问题标记、窗口、版本自检 | 不自动上传 |

## 保持唯一所有权

- 树结构：`simple-mind-map`；
- 撤销/重做：`simple-mind-map` 历史；
- 导图仓库：YeMind `MapRepository`；
- 检查点：YeMind `CheckpointRepository`；
- 当前活动大纲编辑器：一个 Quill；
- 节点备注：一个 `yemindNote`；
- 节点批注：一个 `yemindComments` 数组。
