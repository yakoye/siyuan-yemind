# YeMind v0.9.23 product boundaries

## Included

- Fixed 22px icon slots and 15px proportional artwork for YeMind toolbar and context-menu controls.
- Matching 22/15 geometry for native SiYuan SVG icons used inside YeMind context menus.
- Separate isolated light and dark image documents for all 14 supplied Base64 SVG icons.
- Theme-aware outline hover, selection, disclosure and drag-inside states.
- Higher-contrast dark top-toolbar hover and active states.
- Appearance markers on detached canvas and node context menus.

## Excluded

- No map schema, node UID, hierarchy, relation, image or checkpoint migration.
- No replacement of SiYuan global menu styling outside YeMind-owned menus.
- No changes to the fixed marker, clipart or layout-thumbnail resource catalogs.
- No user data files in the release archive.
