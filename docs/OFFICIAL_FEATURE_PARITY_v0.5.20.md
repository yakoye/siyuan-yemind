# YeMind v0.5.20 官方功能对齐边界

| 功能 | 官方 KMind Zen 0.34.0 | YeMind v0.5.20 | 边界 |
|---|---|---|---|
| 节点快速操作 | 编辑器内核绘制 | 编辑器局部 HTML 浮层 + 上游命令 | 不复制官方私有渲染器 |
| 折叠同步 | 统一文档状态 | 原生 `expand` + `SET_NODE_EXPAND` | Root 本身固定 |
| 结构选择 | 官方结构菜单 | 顶部图标与空白右键子菜单 | 继续使用 YeMind 布局预设 |
| 主题选择 | 官方主题系统 | YeMind 主题预设 + 图标菜单 | 不导入官方私有主题编辑器 |
| 线型选择 | 图标网格 | 当前线型 SVG + 子菜单 | 三种现有线型 |
| 节点样式 | 完整属性面板 | 形状、填充、边框、宽度和文本核心字段 | 使用上游原生样式字段 |
| 概要 | 原生概要 | 原生概要 + 专用图标 + 大纲行 | 范围和括号由上游负责 |
