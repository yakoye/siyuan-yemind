# YeMind Zen v0.5.16 实施计划

日期：2026-07-18

## 目标与架构

基于 v0.5.15，通过思源原生 `Menu`、现有 `YeMindCommands`、`simple-mind-map` 的 `SET_NODE_IMAGE` 与 `NodeImgAdjust` 完成八项交互，不修改数据版本和仓库边界。

## Task 1：建立版本与研究文档

修改：
- `package.json`
- `package-lock.json`
- `plugin.json`
- `AGENTS.md`
- `docs/official-source-interaction-analysis-v0.5.16.md`

步骤：
1. 记录官方画布/节点菜单、禅模式和图片命令证据。
2. 更新基线到 v0.5.16。
3. 不改变插件 ID、存储名和数据 schema。

## Task 2：节点装饰 RED/GREEN

测试：
- `tests/nodeDecorationsSettings.test.ts`
- `tests/v0516NodeChrome.test.ts`

RED：断言激活节点不生成 `.ymz-node-menu-button`，多条批注不生成 `.ymz-node-comment-count`，aria-label 不含数量。

GREEN：修改 `src/core/nodeDecorations.ts`、`src/settings/SettingsStore.ts`、`src/settings/settingsDialogTemplate.ts` 和相关 CSS。

## Task 3：颜色面板 RED/GREEN

测试：
- `tests/richTextToolbar.test.ts`
- `tests/colorPresentation.test.ts`

RED：断言不存在 eyedropper，存在 RGB/HEX，当前格式和重置状态会更新读数。

GREEN：新增 `src/editor/colorPresentation.ts`，修改 `src/editor/RichTextToolbar.ts` 与 `src/styles/index.css`。

## Task 4：大纲删除语义 RED/GREEN

测试：
- `tests/outlineEditing.test.ts`
- `tests/outlineRichTextController.test.ts`

RED：空行 Backspace/Delete 返回 `none`，结构删除不再由键盘处理器触发。

GREEN：修改 `src/editor/outline.ts`，删除 `YeMindEditor.handleOutlineKeydown()` 中隐式 remove 分支。

## Task 5：画布与节点菜单 RED/GREEN

测试：
- `tests/v0516ContextMenus.test.ts`
- 既有菜单可用性和回归测试

RED：要求画布菜单存在、编辑器订阅上游 `contextmenu`、菜单元素 class 和 CSS 最大高度/细滚动条存在。

GREEN：
- 扩展 `src/ui/contextMenu.ts`；
- 扩展 `src/core/commands.ts`；
- 修改 `src/editor/YeMindEditor.ts`；
- 修改 `src/styles/index.css` 和思源类型 shim。

## Task 6：禅模式退出胶囊 RED/GREEN

测试：
- `tests/v0516ZenCapsule.test.ts`
- 既有编辑器模板测试

RED：要求默认紧凑、hover/focus 展开、文字被内层 span 包裹且按钮保持可访问。

GREEN：修改 `src/editor/editorTemplate.ts` 和 `src/styles/index.css`。

## Task 7：图片粘贴、拖入与等比例缩放 RED/GREEN

测试：
- `tests/nodeImageInput.test.ts`
- `tests/v0516ImageIntegration.test.ts`
- 既有图片读取和插件注册测试

RED：识别剪贴板/拖入的 image 文件，忽略非图片；命中节点并写入自然尺寸；只读/无目标不吞事件；编辑器绑定 paste/dragover/drop；继续使用 `NodeImgAdjust`。

GREEN：
- 新增 `src/ui/nodeImageInput.ts`；
- 修改 `src/editor/YeMindEditor.ts`；
- 继续复用 `src/ui/imageFileLoading.ts`；
- 不实现第二套缩放器。
