# YeMind v0.5.20 实施计划

日期：2026-07-20

## 技术栈

- TypeScript
- `simple-mind-map` 0.14.0-fix.3
- SiYuan Plugin API
- Vitest + jsdom
- Vite

## Task 1：建立版本与官方研究文档

文件：`package.json`、`plugin.json`、`src/plugin/constants.ts`、`AGENTS.md`、官方研究文档。

步骤：记录官方边界；版本统一更新为 0.5.20；运行版本一致性测试。

## Task 2：节点快速新增/删除

文件：新增 `src/editor/nodeQuickActions.ts`，修改 `YeMindEditor.ts`、样式和测试。

RED：叶节点只返回新增；非叶节点返回新增和删除；Root 不返回删除；点击调用指定 UID 原生命令。

GREEN：实现局部 SVG 快速控制层并在节点、视图、布局、缩放事件后重新定位。

## Task 3：折叠双向同步

文件：修改 `outline.ts`、`YeMindEditor.ts`、`commands.ts` 和测试。

RED：大纲展开值来自节点数据；折叠隐藏后代；大纲三角调用原生命令；导图数据变化后同步；Root 一级分支可折叠。

GREEN：使用 `setNodeExpandedByUid`，保留活动编辑器稳定 patch。

## Task 4：结构、主题和线型工具

文件：修改 `editorTemplate.ts`、`YeMindEditor.ts`、`contextMenu.ts`、样式和测试；新增 `projectControls.ts`。

RED：顶部不存在删除；三个项目级入口有图标和正确标签；空白菜单包含三个子菜单；节点菜单不包含。

GREEN：使用插件内置 SVG；通过统一回调应用并保存；线型按钮图标随当前值变化。

## Task 5：节点样式面板

文件：新增 `nodeStyle.ts`、`nodeStylePanel.ts`，修改命令、编辑器、模板、样式和测试。

RED：规范化样式 patch；重置清除局部覆盖；多选批量应用；面板输入不触发节点快捷键。

GREEN：增加节点样式命令；用上游样式命令，不可用时受控回退原生节点数据命令；选中回显。

## Task 6：概要入口与编辑

文件：修改命令、菜单、装饰、大纲、编辑器和测试。

RED：专用图标；连续同级选择可添加；创建后定位编辑；大纲包含概要行；删除概要不删覆盖节点。

GREEN：调用 `ADD_GENERALIZATION`，查找新增概要并进入上游编辑。

node --check index.js
