# YeMind v0.5.19 Outline Delete and Collapse Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [x]`) syntax for tracking.

**Goal:** Make empty outline rows deletable, isolate outline folding from map folding, add indent guides, and safely intercept canvas Delete/Backspace without Root error dialogs.

**Architecture:** Keep structural mutations in the existing command adapter. Add editor-local outline collapse state consumed by pure outline rendering helpers, and resolve upstream deletion at `beforeShortcutRun` so Root nodes are filtered before the library command executes.

**Tech Stack:** TypeScript, Quill 2, simple-mind-map 0.14.0-fix.3, Vitest/jsdom, Vite.
