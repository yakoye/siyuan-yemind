# YeMind Zen v0.5.10 实施计划

1. 为诊断自检隔离、待办二态复选框、标签存活检查、安全 resize 和拖拽虚线写失败测试。
2. 将生命周期自检迁移到独立临时存储，增加并发自检锁。
3. 修复 Dock 加载状态与仓库就绪后的强制刷新。
4. 增加零尺寸等待工具，标签挂载和视图切换统一使用安全 resize。
5. 给标签注册表增加陈旧句柄清理。
6. 分离待办复选框二态切换和右键三态命令。
7. 用 YeMindDrag 继承上游 Drag，仅增强目标父节点虚线。
8. 分析两份诊断包并写入验证报告。
9. 跑全部测试、TypeScript、生产构建、ZIP 完整性和解压复检。
