# YeMind v0.9.0 implementation plan

## Task 1: image actions

- Replace the image remove glyph with a trash-can SVG.
- Add a top-left magnifier button to the image adjustment overlay.
- Add a reusable confirmation guard and apply it to both removal entry points.
- Add DOM and command-contract tests.

## Task 2: image lightbox

- Mount the overlay inside `.ymz-editor`.
- Add wheel zoom, scale label, 1:1 reset and close gestures.
- Add helper and DOM interaction tests.

## Task 3: hover badges

- Remove native title attributes from note and comment badges.
- Preserve aria labels and custom hover events.
- Add regression coverage.

## Task 4: palettes and themes

- Define ten named palettes in one registry.
- Add project-style palette selection and persistence.
- Generate matching whole-map theme variants and backgrounds.
- Retain three base themes and internal migration aliases.
- Add persistence, rendering and UI tests.

## Task 5: product cleanup and release

- Update public documentation to the current YeMind identity and SiYuan 3.7.3 baseline.
- Update version metadata to 0.9.0.
- Run all tests, TypeScript, build and entry syntax validation.
- Create a flat release ZIP, re-extract, install dependencies and repeat the full gate.
