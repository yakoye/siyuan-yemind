# YeMind Zen v0.6.8 实施计划

日期：2026-07-21

## Task 1：建立真实宿主 fixture 与失败测试

- 从用户提供的思源全局搜索 HTML 中提取真实结构；
- 验证旧结果错误地出现在搜索输入区；
- 增加结果列表、预览、鼠标、键盘、重绘测试。

## Task 2：重构搜索模型与渲染

- 增加匹配排序；
- 增加搜索词安全高亮；
- 结果条目改为原生列表结构；
- 增加来源标签与右侧路径。

## Task 3：增加大纲预览

- 查找目标节点路径；
- 构建有上下文的只读大纲；
- 使用覆盖层而不是销毁原生预览；
- 普通结果选中时恢复原生预览。

## Task 4：统一鼠标与键盘操作

- 单击预览；
- 双击/Enter 打开；
- Alt 操作右侧打开；
- 上下键、PageUp/PageDown 与原生结果衔接；
- 失败降级为打开导图。

## Task 5：宿主重绘恢复

- MutationObserver 检测列表和预览重建；
- 合并自身 DOM 变更；
- 防止重复注入和事件重复注册。

## Task 6：插件打开接口

- `openMap` / `openMapAtNode` 支持可选 `position: right`；
- 已打开标签继续复用；
- 首次打开支持右侧分屏。

## Task 7：全量验证和发布

```bash
npm test
npm run check
npm run build
node --check index.js
npm audit --omit=dev
```

生成 ZIP 后完整解压并重新执行安装、测试、检查和构建。
