# YeMind Zen v0.5.11 实施计划

## 1. 启动链路回归测试

- 新增纯函数启动协调器测试。
- 证明 bootstrap 在所有界面注册之前开始。
- 证明任一同步注册抛错时，后续注册仍执行，ready 仍指向真实 bootstrap。
- 证明注册错误带步骤名上报。

## 2. 启动链路实现

- 新增 `pluginStartup.ts`。
- 将 `YeMindZenPlugin.onload()` 改为先启动 bootstrap，再逐项隔离注册。
- 增加 `registration-started/completed/failed` 诊断事件。
- 保留事件总线卸载安全性。

## 3. 大纲模型与键盘测试

- 扩展大纲行数据：根节点标记。
- 测试可编辑 HTML 语义、UID、层级和只读状态。
- 测试 Enter、Tab、Backspace/Delete、ArrowUp/Down、Escape 的动作解析。
- 测试删除聚焦回退和新节点 UID 行为。

## 4. 上游命令适配

- 在命令适配器加入按 UID 选择、改名、增加同级/子级、删除能力。
- 所有结构操作调用 `simple-mind-map` 原生命令。
- 增加命令适配器单元测试，禁止直接写树结构。

## 5. 大纲 DOM 编辑器

- 将只读按钮改为 treeitem 行与输入框。
- 绑定焦点、点击、input、keydown、blur。
- 在重新渲染后恢复目标焦点与编辑选择。
- 与画布节点激活事件双向同步。
- 只读模式同步禁用输入。

## 6. 样式与视图

- 分屏保持右侧大纲布局；纯大纲占满工作区。
- 对齐用户截图中的层级、圆点、缩进、选中与编辑状态。
- 保证长文本换行而非强制单行截断。

## 7. 版本、文档与验证

- 版本升级到 0.5.11。
- 更新 CHANGELOG、README 和验证报告。
- 运行全部测试、TypeScript、生产构建、入口语法与 ZIP 完整性检查。
- 输出安装包和真实桌面端手工回归清单。
