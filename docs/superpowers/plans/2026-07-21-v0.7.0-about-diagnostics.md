# YeMind Zen v0.7.0 实施计划

1. **RED：关于与发布身份**
   - 新增测试，要求 About 页展示 0.7.0、版本字段和三个动作。
   - 新增 `releaseInfo.ts`，统一运行时和构建元数据。
2. **GREEN：关于页面**
   - 修改设置模板、交互和样式。
   - 接入复制版本、打开诊断和导出诊断。
3. **RED：结构化诊断包**
   - 测试要求摘要、环境、版本、时间线、搜索、编辑器、回归、错误、标记和 DOM 状态文件。
4. **GREEN：诊断 schema v3**
   - 扩展 DiagnosticsService、自检和诊断窗口。
   - 默认脱敏，保留兼容汇总文件。
5. **RED/GREEN：全局搜索事件链**
   - 测试查询、计数、挂载、选择、Enter、关闭和打开事件。
   - 在搜索适配器、插件标签和节点聚焦中记录状态。
6. **发布收口**
   - 更新版本、语义化版本文档、README、CHANGELOG、矩阵和迁移状态。
   - 运行 `npm test`、`npm run check`、`npm run build`、`node --check index.js` 和生产依赖审计。
   - 打包后完整解压并重复安装、测试、检查和构建。
