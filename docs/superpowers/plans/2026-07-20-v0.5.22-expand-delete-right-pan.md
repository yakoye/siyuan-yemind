# YeMind Zen v0.5.22 实施计划

日期：2026-07-20

## Task 1：失败测试

新增 `tests/v0522ExpandDeletePan.test.ts`：

- 折叠节点的 live children 为空、持久化 children 非空时仍可展开；
- Root 使用相同规则；
- 大纲编辑宿主及后代属于可编辑目标；
- 右键移动阈值、增量位移、菜单抑制和静止右击；
- pan/select 两种画布模式均验证。

## Task 2：展开命令修复

修改 `src/core/commands.ts`：

- 使用持久化子树判断是否存在子节点；
- 保留上游 `SET_NODE_EXPAND` 作为唯一最终命令。

修改 `src/editor/YeMindEditor.ts`：

- 大纲三角直接调用明确展开/折叠命令，不先执行 `GO_TARGET_NODE`。

## Task 3：编辑删除隔离

修改 `src/editor/shortcuts.ts`：

- 将 `[data-outline-editor]` 识别为编辑目标。

修改 `src/editor/YeMindEditor.ts`：

- Root 捕获阶段保护活动大纲编辑会话；
- 大纲冒泡阶段拦截 Backspace/Delete 向 window 传播；
- 保持 Quill 默认删除和空节点结构删除两条独立路径。

## Task 4：右键平移

新增 `src/editor/canvasRightDrag.ts`：

- 纯手势状态机；
- 画布控制器；
- pan 模式增量平移；
- select 模式复用上游平移；
- 拖动菜单抑制。

修改 `src/styles/index.css`：

- 拖动期间统一 `grabbing` 光标。

修改 `src/editor/YeMindEditor.ts`：

- 创建/销毁控制器；
- 节点和空白菜单打开前消费拖动菜单状态。
