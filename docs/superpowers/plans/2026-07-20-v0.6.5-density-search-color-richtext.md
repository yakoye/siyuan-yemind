# YeMind Zen v0.6.5 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 修复整图密度、搜索定位、节点颜色和画布富文本回归。

**Architecture:** 保留 simple-mind-map 的结构、历史和渲染权威；新增的间距仍通过 themeConfig 覆盖，搜索定位通过标签挂载状态排队，颜色交互复用共享色板，富文本可见性在上游编辑器创建后同步真实节点样式。

**Tech Stack:** TypeScript, simple-mind-map, Quill, SiYuan Plugin API, Vitest, Vite.
