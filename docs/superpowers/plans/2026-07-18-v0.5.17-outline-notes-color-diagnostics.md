# YeMind v0.5.17 实施计划

日期：2026-07-18

## 技术栈

TypeScript、Vitest/jsdom、Quill 2、simple-mind-map 0.14.0-fix.3、SiYuan Plugin API、Vite。

## 任务 1：版本与设计基线

修改：`package.json`、`plugin.json`、`src/plugin/constants.ts`、README、CHANGELOG、FEATURE_MATRIX。

测试先行：更新 `tests/pluginIdentity.test.ts`，增加运行时/manifest 一致性测试。

命令：`npx vitest run tests/pluginIdentity.test.ts`，先失败后通过。

## 任务 2：待办、模糊、π、禅胶囊

修改：`src/content/nodeContentState.ts`、`src/ui/nodeContentMenu.ts`、`src/ui/contextMenu.ts`、`src/editor/RichTextToolbar.ts`、`src/editor/editorTemplate.ts`、`src/styles/index.css`。

测试先行：新增/修改 todo、toolbar 和 zen 测试，确认旧文案失败。

## 任务 3：颜色解析、可编辑输入与事件隔离

修改：`src/editor/colorPresentation.ts`、`src/editor/RichTextToolbar.ts`、`src/editor/shortcutSafety.ts`、样式。

测试先行：

- HEX/RGB 双向解析；
- 非法或未完成输入不应用；
- input 的 key/input/paste/composition 不传播；
- 手动输入保留面板，色块点击关闭。

命令：`npx vitest run tests/colorPresentation.test.ts tests/richTextToolbar.test.ts tests/richTextIsolation.test.ts`。

## 任务 4：备注数据、编辑和悬停预览

新增：`src/content/nodeNoteState.ts`、`src/ui/nodeHoverPreview.ts`。

修改：`src/model/types.ts`、`src/core/commands.ts`、`src/core/nodeDecorations.ts`、`src/ui/nodeContentDialogs.ts`、`src/ui/contextMenu.ts`、`src/editor/YeMindEditor.ts`、样式。

测试先行：备注规范化、命令写回、图标、悬停延迟/保持、批注菜单 SVG。

## 任务 5：大纲稳定编辑与增量渲染

新增：`src/editor/outlinePatch.ts`、`src/editor/outlineEditingPolicy.ts`。

修改：`OutlineRichTextController.ts`、`outline.ts`、`YeMindEditor.ts`。

测试先行：

- 活动行文本提交不替换 DOM；
- IME 期间不 flush/结构操作；
- Backspace/Delete 不转为删除节点；
- 根 Enter 添加子节点；
- 普通编辑、自动保存、selection 恢复；
- 上下键只在边界跨行。

## 任务 6：逐行拖拽

新增：`src/editor/outlineDrag.ts`。

修改：`outline.ts`、`YeMindEditor.ts`、`commands.ts`、样式和类型声明。

测试先行：拖拽意图三区、非法目标、命令映射、只读、flush 失败中止、每行拖拽手柄。

## 任务 7：诊断增强

修改：`DiagnosticsRecorder.ts`、`DiagnosticsService.ts`、`diagnosticsDialog.ts`、`selfCheck.ts`、`YeMindEditor.ts`。

测试先行：问题标记、窗口摘要、view-change 聚合、编辑事件、节点数变化、版本一致性失败。

## 任务 8：规格与质量审查

逐条核对设计；检查无第二套结构/历史；检查事件监听销毁、悬停计时器、拖拽 dataTransfer、HTML 清理和敏感数据脱敏。

## 任务 9：完整验证与打包

运行：

```bash
npm test
npm run check
npm run build
node --check index.js
npm audit --json
```

打包时排除 `node_modules/`、临时 `dist/`、`.git/`。执行：

```bash
unzip -t /mnt/data/siyuan-yemind-zen-v0.5.17.zip
node --check /mnt/data/yemind-v0517-package-check/index.js
```

输出：

- `siyuan-yemind-zen-v0.5.17.zip`
- `YeMind-v0.5.17-verification.md`
- `docs/verification-v0.5.17.md`
- `docs/official-source-outline-notes-analysis-v0.5.17.md`
