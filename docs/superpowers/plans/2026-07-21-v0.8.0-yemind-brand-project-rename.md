> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the product icon and rename the public product and engineering project to YeMind while retaining the legacy SiYuan plugin ID for data-compatible upgrades.

**Architecture:** Centralize current identity constants, rename the plugin source entry/class and package metadata, and generate deterministic transparent PNG assets from the supplied source logo. Keep `plugin.json.name` as a legacy compatibility identifier, but remove Zen from all current visible UI and release artifacts.

**Tech Stack:** TypeScript, Vite, Vitest, Python/Pillow for deterministic icon generation, SiYuan Plugin API.
