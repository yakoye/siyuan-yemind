# YeMind v0.7.1 全局搜索打开修复设计

## 根因
真实诊断显示所有导航均停在 `close-request`，随后抛出 `close2.click is not a function`。思源当前窗口中的关闭控件可能是 SVG 或其他 Element，不保证具有 `HTMLElement.click()`。

## 设计
- 关闭适配器只在 `click` 为函数时调用；否则向实际 Element 派发可冒泡的合成 click。
- click 激活失败时依次向输入框、对话框、document 和 window 派发 Escape。
- 关闭搜索失败不得阻断 `onOpen`。
- Enter、双击结果和预览“打开导图”共用同一导航函数。
- 诊断自检将超过 1.5 秒仍停在 Enter、关闭或打开阶段的会话判定为失败。

## 兼容性
不修改导图数据、搜索索引或存储格式，属于 Patch 修复。
