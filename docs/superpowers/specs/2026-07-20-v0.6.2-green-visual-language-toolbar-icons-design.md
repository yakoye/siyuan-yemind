# YeMind Zen v0.6.2 绿色视觉语言与工具图标设计

日期：2026-07-20

## 目标

1. 所有 YeMind 自有选中、聚焦和编辑状态统一使用插件绿色 `#176B50`，不再跟随思源默认蓝色。
2. 画布操作模式在工具栏和设置页统一称为“选（选择优先）”与“拖（拖动优先）”。
3. 新安装默认采用“选”，即左键框选、右键拖动画布；默认态不显示高亮。
4. 左侧工具栏只保留历史、撤销和重做，并全部使用 SVG 图标。
5. “选/拖”移动到底部锁定按钮左侧。
6. 搜索、锁定、禅模式和禅模式退出入口使用统一 SVG 图标。

## 交互约束

- “选”是默认模式，按钮不高亮。
- 切换为“拖”后按钮使用绿色选中态，文字变为“拖”。
- 设置页选项与工具栏提示必须使用相同名称和操作说明。
- 禅模式图标使用插件自有的冥想人物线稿，不复制用户提供图片中的水印或商业素材。
- 图标全部使用 `currentColor`，浅色主题下呈深色，深色主题下自动适配。

## 绿色状态范围

- 顶部导图/分屏/大纲选中按钮；
- 左侧及底部工具按钮的激活状态；
- 大纲当前行、编辑框焦点和文字选区；
- 画布节点激活轮廓；
- 设置导航、富文本按钮、节点样式按钮；
- 拖拽落点、分屏分隔条、颜色输入焦点和备注编辑焦点。

## 保持不变

- 导图结构、历史、检查点和持久化格式不变；
- 右键拖动画布、富文本选择、大纲拖拽和折叠回归继续作为发布门禁；
- “整理布局”功能仍保留在右键菜单和命令层，只移除左侧重复入口。
