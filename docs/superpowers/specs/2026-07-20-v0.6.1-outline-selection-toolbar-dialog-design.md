# YeMind Zen v0.6.1 大纲文字选择、工具栏重排与弹窗布局设计

日期：2026-07-20

## 目标

本版完成四组已经明确的需求：

1. 修复分屏/纯大纲编辑状态下拖选文字被整行拖拽抢占；
2. 精简并重排顶部、左侧和底部工具栏；
3. 将备注和批注关闭按钮放入各自弹窗右上角；
4. 统一 YeMind 品牌块和导图标题块为插件图标绿色背景；
5. 版本升级为 v0.6.1，并把本轮与此前高频回归行为纳入长期测试集。

## 现状与根因

### 大纲文字选择

当前整行拖拽在 `pointerdown` 时记录会话。事件落在 `[data-outline-editor]` 内时，只要求长按 240ms 且移动超过 6px，就会转为结构拖拽。Quill 编辑状态下用户拖选一段文字也满足这个条件，因此选择被中断并出现拖拽幽灵。

### 工具栏重复

顶部、左侧和底部存在重复入口：

- 顶部与左侧重复撤销/重做；
- 顶部与底部重复搜索；
- 左侧与底部重复适配视图；
- 顶部仍保留节点新增与检查点；
- 线型只有图标，没有清晰文字；
- 节点样式仅能从节点菜单打开。

### 备注/批注关闭位置

SiYuan `Dialog` 的原生关闭按钮由宿主标题栏控制，真实桌面主题下可能出现在内容区外侧。备注和批注需要隐藏宿主关闭图标，使用插件自己的内容区标题栏和关闭按钮。

## 方案比较

### 方案 A：彻底禁止从大纲正文区域拖动

- 优点：文字选择最稳定。
- 缺点：整行大部分区域均是正文，用户几乎无法拖动节点。

### 方案 B：增加新的拖拽把手

- 优点：选择和拖拽边界清晰。
- 缺点：与用户已经明确要求去掉六点把手冲突，并增加视觉噪声。

### 方案 C：按编辑状态隔离正文拖拽（采用）

- 未进入 Quill 编辑时，整行正文仍可长按后拖动；
- 当前行一旦进入编辑状态，`[contenteditable=true]`、`.ql-editor` 及活动编辑宿主中的 pointer 手势永远只用于光标和文字选择；
- 折叠三角、按钮、链接、输入控件继续作为交互控件，不启动结构拖拽；
- 行缩进空白区仍可直接拖动。

该方案保留整行拖动，同时满足编辑时稳定选择文字。

## 工具栏最终布局

### 顶部

保留：

- YeMind；
- 导图 / 分屏 / 大纲；
- 搜索；
- 结构；
- 主题；
- 线型（图标 + 文字）；
- 节点样式（图标 + 文字）；
- 保存状态。

移除：

- 检查点；
- 撤销 / 重做；
- `+子` / `+同`。

### 左侧

保留：

- 历史（原“重置视图”位置，打开检查点/历史菜单）；
- 整理布局；
- 框选/平移模式；
- 撤销；
- 重做。

移除：

- 适配视图；
- 重置视图。

### 底部

保留：

- 导图标题；
- 统计；
- 选择数量；
- 适配视图；
- 只读；
- 禅模式；
- 缩放；
- 全屏；
- 帮助。

移除：

- 搜索。

## 节点样式入口

顶部增加 `data-action="node-style"` 按钮。只有存在可用节点选择时启用，点击复用现有 `NodeStylePanel.show()`，不建立第二套面板。

## 历史入口

左侧“历史”按钮继续使用现有 `openCheckpointMenu()`：

- 创建检查点；
- 管理检查点和恢复历史。

因此本版只是移动入口，不改变检查点数据结构或恢复机制。

## 备注与批注弹窗

- `Dialog` 使用 `hideCloseIcon: true`；
- 弹窗内容顶部增加 `.ymz-node-dialog__header`；
- 左侧显示“备注/批注”标题，右侧显示 `×`；
- 点击 `×` 仅关闭弹窗，不自动保存未确认的备注；
- 备注原有保存/取消按钮继续工作；
- 批注现有即时持久化逻辑保持不变；
- 宿主原生空标题栏在这两类弹窗中隐藏。

## 品牌颜色

使用项目已有 `--ymz-green`，与 `icon.png` 背景一致：

- `.ymz-brand`；
- `.ymz-status-title`。

文字继续使用白色，保持对比度。

## 回归测试

新增永久回归矩阵，至少覆盖：

- 活动 Quill 编辑区拖选文字永远不启动大纲拖拽；
- 未编辑大纲行仍可长按拖动；
- 顶部、左侧、底部入口唯一性；
- 节点样式顶部入口调用现有面板；
- 历史入口调用检查点菜单；
- 备注/批注自有关闭按钮和隐藏宿主关闭图标；
- 品牌和标题使用 `--ymz-green`；
- Root/普通节点折叠数字与三角展开；
- 大纲 Delete/Backspace 编辑隔离；
- 右键拖动画布不弹菜单；
- 画布与大纲富文本选区工具栏。

## 非目标

- 不改变检查点存储格式；
- 不重新设计节点样式面板；
- 不新增拖拽把手；
- 不改动导图快捷新增 `+ / −` 逻辑；
- 不改动备注或批注的数据模型。
