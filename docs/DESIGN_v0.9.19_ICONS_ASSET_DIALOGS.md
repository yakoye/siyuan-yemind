# YeMind v0.9.19 design: icons, asset dialogs and checkpoint access

The supplied SVG artwork is normalized to `currentColor` while retaining original paths. Marker categories remain pinned above one continuous scroll surface. Clipart uses a complete filtered result set with lazy image loading rather than pagination. Checkpoint creation is placed in the manager so the toolbar entry is a single reliable destination. The outline drop marker uses the same branch-center geometry as triangles and leaf squares.
