# YeMind Zen v0.6.3 官方交互适配边界

日期：2026-07-20

## 本版参考

本版继续参考用户提供的 KMind Zen 0.34.0 在内容浮层、紧凑面板、菜单分组和编辑器局部 surface 上的交互原则，同时保留 YeMind 当前 `simple-mind-map` 数据、命令、历史、检查点和持久化边界。

## 已适配

- 选择/拖动画布使用图标优先的紧凑控制，并在设置页提供一致入口。
- 节点内容预览围绕触发入口进行多方向避让，而不是固定覆盖在节点右侧。
- 顶部和底部项目工具作为编辑器局部、单行、可滚动 surface，适应窄标签页。
- 备注/批注使用内部标题栏和紧凑内容卡片。
- 节点样式从顶部和右键菜单进入同一个 YeMind 面板与上游样式命令。
- 节点菜单使用清晰分组，根菜单不再承载两层滚动。

## 保持不变

- 不引入 KMind 私有文档模型、React 状态树或第二套历史。
- 不改变节点内容字段、备注/批注数据、节点样式字段或导图版本。
- 不替换 `simple-mind-map` 的结构、拖拽、概要、样式和撤销命令。
- 不把悬停预览位置写入导图数据。

## 自动验证边界

自动测试覆盖几何避让、图标同源、响应式 CSS、对话框结构、菜单组织与单滚动容器；Windows SiYuan 3.7.2 中的真实字体、宿主菜单尺寸、屏幕缩放和极窄标签页仍需桌面验收。
