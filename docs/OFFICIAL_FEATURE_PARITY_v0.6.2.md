# YeMind Zen v0.6.2 官方交互适配边界

日期：2026-07-20

## 本版适配

- 继续沿用官方 KMind 的项目级工具分组思想，但根据 YeMind 的精简界面重新安排顶部、左侧和底部工具位置。
- 选中、聚焦和编辑状态使用 YeMind 自有绿色视觉语言，不依赖宿主默认蓝色主题变量。
- 画布模式继续调用 `simple-mind-map` 的 `useLeftKeySelectionRightKeyDrag` 等原生能力，只统一用户可见名称为“选/拖”。
- 历史、撤销、重做、搜索、锁定和禅模式使用插件自有 SVG，不导入官方私有图标包。
- 禅模式图标为自制冥想人物线稿，只参考用户描述的语义，不复制带水印的商业素材。

## 保持的内核边界

本版没有引入：

- 第二套导图数据结构；
- 第二套撤销历史；
- 官方私有文档模型；
- 新的地图持久化格式；
- 额外的图标运行时或外部图片依赖。

所有地图编辑、选择、拖动、折叠、历史和保存继续由现有 YeMind 与 `simple-mind-map` 适配层负责。
