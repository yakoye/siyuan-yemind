# YeMind v0.9.0 product boundaries

YeMind owns its product interface, map repository, settings, checkpoints, diagnostics, search integration, outline behavior and editor coordination. The rendering engine remains an explicit third-party dependency documented in `THIRD_PARTY_NOTICES.md`.

v0.9.0 changes image interaction and appearance configuration only. Existing map structure, node UIDs, command history and storage files remain compatible.
