# YeMind v0.9.20 product boundaries

- Host baseline: SiYuan 3.7.3.
- The release changes command artwork, marker/clipart dialog presentation, and structural drag intent.
- No map schema, UID, node text, relation, image, checkpoint, or settings migration is introduced.
- Cross-root direction persistence is limited to root children in bidirectional mind-map layouts.
- The resource-excluded ZIP intentionally omits the large `assets/` directory retained by the user locally.
